# kuzeykara.com

A personal portfolio site showcasing projects and blog posts. It serves as a central place for my work and writing.

## Stack

- **Astro** — Static site generator with content collections for blog and projects
- **Tailwind CSS** — Styling (via `@tailwindcss/vite`)
- **Vite** — Build tool and dev server
- **TypeScript** — Typing across the codebase
- **Fonts** — Inter (variable) and JetBrains Mono (variable) from Fontsource

## Commands

- `npm run dev` — Start the dev server
- `npm run build` — Production build
- `npm run preview` — Preview the production build locally
